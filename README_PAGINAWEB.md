## 📁 Estructura

```
PaginaWeb/
├── index.html       ← página principal
├── sqlquest.html    ← juego interactivo de SQL
├── styles/          ← hojas de estilo
├── scripts/         ← lógica JavaScript
└── imagenes/        ← assets visuales
```

## 🎮 SQLQuest

Incluye **SQLQuest** — un juego interactivo para aprender y practicar SQL de forma divertida. Ideal para estudiantes de bases de datos y desarrolladores que quieren repasar sus habilidades.
